Background:

'Albert in City' is a simple web interactive developed using VisiNeat and HTML5. VisiNeat is a JavaScript library that provides powerful functions to build and render 3D graphics in the web browser. It is built on top of WebGL. VisiNeat was developed by Dr. Angelos Barmpoutis at the University of Florida. I made this project for a class taught by Dr. Barmpoutis at UF to demo how JavaScript can be used to develop interactive 3D applications.
