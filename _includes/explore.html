The Problem:

The 'Explore Programs' tool is a strategically important tool for the Undergraduate Education division at IU. Through this tool both new students and existing students discover programs that match their interests and more importantly, will find out which courses can accelerate their degree progress at IU. While the tool existed, the design of the tool was outdated, hard to use, failing various accessibility and usability standards. I re-designed the application which is a complete transformation of the application that was existing.

My role:

I worked as the Information Architect , lead User Interface Designer and contributed to developing the interactions in code using HTML + CSS + jQuery. Following are some of the standout features in the re-design of the explore programs tool

1. Accessibility and Usability First Approach:
2. Accelerating decision making through providing clarity
3. Information flow based on the student's experience flow
4. The tool as dynamic as the needs of today's students
